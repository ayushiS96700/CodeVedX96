
# Utility Usage Prediction Tool using Machine Learning

## Project Overview

The **Utility Usage Prediction Tool** is a Python-based console application that predicts electricity usage based on historical data. It uses a **Linear Regression** machine learning model to estimate future electricity bills using the month number and the number of occupants.

The application stores data in a CSV file, allows users to add or update records, and trains the model each time a prediction is requested.

---

## Features

- View existing utility usage data
- Add new utility usage records
- Update existing records
- Store data in a CSV file
- Predict future electricity bills using Machine Learning
- Input validation and error handling
- Simple menu-driven console interface
