# 🎮 Hangman Game (Python)

## 📌 Project Overview
This is a simple command-line Hangman Game developed using Python. The player selects a category, guesses letters to reveal the hidden word, and tries to complete the word before running out of attempts.

## ✨ Features
- Multiple Categories
  - Programming
  - Animals
  - Countries
- Random word selection
- ASCII Hangman graphics
- One-time hint option
- Tracks correct and wrong guesses
- Win/Loss score tracking
- Play Again option
- Input validation

## 🛠️ Technologies Used
- Python 3
- Random Module

## 📂 Project Structure

Hangman-Game/
│── hangman.py
│── README.md
└── requirements.txt

## ▶️ How to Run

1. Clone the repository:

git clone <repository-link>


2. Move to the project folder:

cd Hangman-Game


3. Run the program:

python main.py


## 🎯 Game Rules
- Select a category.
- Guess one letter at a time.
- You have 6 incorrect attempts.
- Type **hint** once to reveal one letter.
- Guess the complete word before the hangman is completed.
